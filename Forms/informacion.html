1 Anthony Joshua Avila Laguna Pedidos Proveedor

La funcion CargarPedidos es la que muestra todos los datos que esten en la base de datos. La funcion AgregarPedidos hace que podamos agregar más registros a la base de datos desde la pagina. La funcion CargarPedido es la encargada de colsultar un solo registro dependiendo en este caso del ID. La funcion ActualizarPedidos como su nombre lo dice, nos permitira actualizar los datos del registro seleccionado. Y la funcion EliminarPedidos es la que va a eliminar el registro seleccionado en base a su ID. Cada funcion esta enlazada a una URL con la que puede hacer su ejecucion correspondiente.

2 Isaac David Luque Medina Materiales

En la función CargarMateriales se encarga de mostrar todos los datos que se encuentran en la tabla materiales de la base de datos, AgregarMaterial se encarga de agregar un nuevo registro, CargarMaterial trae un solo registro seleccionado que se colocará en los forms inputs respectivos para ser actualizados, ActualizarMaterial hace referencia a la actualización de dicho registro seleccionado, EliminarMaterial elimina un solo registro por medio de su ID, cada una de estas funciones para poder hacer dicha acción tienen un url específica y un método HTTP de acuerdo a la API REST.

La función CargarFacturas se encarga de mostrar todos los datos que se encuentran en la tabla de factura_cliente de la base de datos my sql, AgregarFactura se encarga de ingresar un nuevo registro, CargarFactura traera un solo registro seleccionado por el cliente que se colocará en los espacios respectivos para ser actualizados, ActualizarFactura realizara la actualización de dicho registro seleccionado por el cliente, EliminarFacrura elimina un solo registro por medio de su ID, cada de las funciones tiene asignada su propia url para ejecutar su accion correspondiente.

4 Scarleth Abigail Baquedano Padilla Socios Negocio

La función CargarSocios es el encargado de mostrar todos los registros que se encuentran en la base de datos de MySQL esto ayudándose de la solicitud de una API, es decir una URL, AgregarSocio, como su nombre lo indica es quien se encarga se agregar un socio a la base de datos, CargarSocio hace la consulta de un solo registro, dependiendo del ID que se solicite cuando damos uso de la función ActualizarSocio dando click en el botón editar, esto se hace para que traiga los resultados de un sólo registro en los label's respectivos y actualizarlos, finalmente EliminarSocio elimina un registro a través de su respectivo ID. Cada una de estas funciones se vale de una URL especifica para realizar las acciones solicitadas.
